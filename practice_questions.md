# What will be the output of the following code blocks?

```python
x = 10
if x > 5:
    print("Hello")
elif x < 15:
    print("World")
else:
    print("Python")
```
a) Hello 

b) World

c) Python

d) Hello World Python

```python
for i in range(3):
    print(i)
```

b) 1 2

c) 0 1

d) 3 

```python
x = 5
while x > 0:
    print(x)
    x -= 1
```

b) 4 3 2 1

c) 5

d) 1 2 3 4 5
